Privacy Policy for OptiScaler Manager

Effective Date: January 2025

Last Updated: January 2025

1. Introduction

OptiScaler Manager ("the App", "we", "our") is a game optimization utility application for Windows that helps users manage upscaling mods (DLSS, FSR, XeSS) for PC games.

This Privacy Policy explains what information we collect, how we use it, and your rights regarding your data.

2. Information We DO NOT Collect

Zero Data Collection

OptiScaler Manager does NOT collect, transmit, or store:

  • Personal information (name, email, address)
  • Usage analytics or telemetry
  • Crash reports (stored locally only)
  • Game library information
  • System information
  • IP addresses or device identifiers
  • Browsing history or online activity
  • Payment or financial information

3. Information Stored Locally

The App stores configuration data only on your local device:

3.1 Application Settings

Stored in: %LocalAppData%\OptiScaler

  • Global Settings: Your preferred upscaler, DLL injection method, GPU preferences
  • Game List: Paths to your installed games (scanned from common game platforms)
  • Mod Configurations: Settings for each game (upscaler, quality preset, etc.)
  • App Preferences: UI theme, launch count for review prompts

3.2 Crash Logs

Stored in: %LocalAppData%\OptiScaler\CrashLogs

  • Generated only when the app crashes unexpectedly
  • Contains exception details and stack traces
  • Stored locally - never transmitted
  • You can view or delete these logs at any time
  • Automatically cleaned up (keeps last 10 logs)

3.3 Downloaded Mods

Stored in: %LocalAppData%\OptiScaler Manager\Mods

  • Upscaling mod files downloaded from GitHub
  • Cached for offline installation
  • You can delete these files to free up space

4. Network Activity

The App connects to the internet only for:

4.1 Mod Downloads

  • Source: GitHub public repositories (github.com/cdozdil/OptiScaler)
  • Purpose: Download latest upscaling mod files
  • Data Sent: HTTP request to GitHub API (public endpoint, no authentication)
  • Data Received: Mod archive files (.7z)

4.2 Microsoft Store Integration

  • Rate & Review Prompt: Uses Windows Store API to open review page
  • Purpose: Allow users to rate the app (optional, user-initiated)
  • Data Sent: None - just launches Store app

No Tracking, No Analytics

OptiScaler Manager does NOT use:

  • Google Analytics or similar services
  • Application Insights or telemetry
  • Error reporting services (e.g., Sentry, Raygun)
  • Advertising networks or trackers
  • Social media integrations

5. File System Access

The App requires access to your file system for its core functionality:

5.1 Read Access

  • Game Directories: Scans common game installation folders to detect installed games
  • Configuration Files: Reads OptiScaler.ini files to load settings
  • Game Metadata: Checks for existing mod installations

5.2 Write Access

  • Mod Installation: Copies DLL and INI files to game directories
  • Configuration Saves: Writes OptiScaler.ini files
  • Local Settings: Saves app configuration to AppData

Broad File System Access

The App uses the broadFileSystemAccess capability to:

  • Scan game folders across all drives (C:, D:, etc.)
  • Install mods to protected directories (Program Files)
  • Support games in custom installation paths

This access is used exclusively for mod installation - no files are accessed outside of game directories and AppData.

6. Third-Party Services

6.1 GitHub

  • Purpose: Download upscaling mod files from public repositories
  • Data Collected by GitHub: IP address, download timestamp (per GitHub's privacy policy)
  • Our Control: None - GitHub manages their own data
  • GitHub Privacy Policy: https://docs.github.com/en/site-policy/privacy-policies

6.2 Microsoft Store

  • Purpose: App distribution, updates, ratings
  • Data Collected by Microsoft: Download/install metrics (per Microsoft's privacy policy)
  • Our Control: None - Microsoft manages Store analytics
  • Microsoft Privacy Policy: https://privacy.microsoft.com/privacystatement

7. Data Security

Since we don't collect or transmit personal data, there is minimal security risk. However:

  • Local Storage: Your settings are stored in Windows protected AppData folder
  • No Cloud Sync: Settings are not synchronized or backed up online
  • No Accounts: No login, no passwords, no user accounts
  • Open Source Mods: All installed mods are from verified public repositories

8. Children's Privacy

OptiScaler Manager is rated PEGI 3 / ESRB Everyone. The App:

  • Does not knowingly collect data from children
  • Does not require parental consent (no data collection)
  • Contains no age-restricted content
  • Is a utility tool, not directed at children

9. Your Rights

Since no personal data is collected, traditional data rights (access, deletion, portability) don't apply. However, you can:

  • View Local Data: Navigate to %LocalAppData%\OptiScaler
  • Delete All Data: Uninstall the app to remove all local files
  • Export Settings: Copy JSON files from AppData (feature coming soon)
  • Opt-Out of Review Prompts: Click "Don't Ask Again" when prompted
