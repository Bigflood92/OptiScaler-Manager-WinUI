# ?? Screenshot Guide for Microsoft Store

## Required Screenshots

Microsoft Store requires **at least 1 screenshot**, but 4-8 is highly recommended for better conversion.

### Recommended Size
- **1920x1080** (Full HD) - Most common
- **3840x2160** (4K) - Optional for high-quality displays

## ?? How to Capture

### Method 1: Windows Snipping Tool
1. Press `Win + Shift + S`
2. Select window/area
3. Save as PNG

### Method 2: In-App (Recommended)
1. Set window to exactly 1920x1080
2. Press `Win + PrintScreen`
3. Find in `Pictures\Screenshots`

### Method 3: Professional (Best Quality)
1. Use **ShareX** (free tool)
2. Configure capture region to 1920x1080
3. Set PNG format, max quality
4. Hotkey capture for speed

---

## ?? Post-Processing Tips

### Optional Enhancements:
1. **Add subtle drop shadow** to windows (makes them pop)
2. **Highlight UI elements** with colored outlines (optional)
3. **Add annotations** like arrows pointing to key features (Store allows this)
4. **Ensure consistent window size** across all screenshots

### Tools:
- **Paint.NET** (free) - Basic editing
- **GIMP** (free) - Advanced editing
- **Figma** (free) - Professional annotations
- **Canva** (free) - Add borders/backgrounds

---

## ?? Don't Include:
- Personal information
- Real game executables paths with your username
- Error messages or crashes
- Unfinished UI elements
- Low-quality or blurry images
- Copyrighted game assets (logos are OK in context)

---

## ?? Screenshot Checklist

Before uploading to Store:
- [ ] All images are 1920x1080 (or 3840x2160)
- [ ] PNG format (not JPG)
- [ ] File size under 5 MB each
- [ ] No personal data visible
- [ ] UI is clean and professional
- [ ] Text is readable even when thumbnails shown
- [ ] Consistent theme/style across all shots
- [ ] Showcase key features (not just generic windows)
- [ ] Include variety (different pages/dialogs)

---

## ?? Screenshot Order in Store

Microsoft Store displays screenshots in order, so arrange them strategically:

1. **Hero shot** (Games Library) - Makes them want the app
2. **Easy to use** (Presets) - Shows simplicity
3. **Powerful** (Configuration) - Shows depth
4. **Results** (Installed mods) - Shows value
5-8. **Additional features** - Nice to have
